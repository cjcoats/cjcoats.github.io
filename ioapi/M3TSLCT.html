m3tools Program M3TSLCT

Usage 



 


 Summary 


Program M3TSLCT takes an ASCII set of dates-and-times
(IN_TIMES, as from program  Program M3MXFIND) and uses it to select
sets of variables for those dates-and-times from a specified input file,
and write them to a gridded output file.



For example, given an hourly-step gridded met-file, and a daily-stepped
ASCII IN_TIMES for daily-max ozone, the program would
construct a daily-stepped gridded meteorology file that shows the
gridded meteorology at which the daily-max ozone occurred.
